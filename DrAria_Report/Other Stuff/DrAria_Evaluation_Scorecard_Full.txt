About This Scorecard
Dr. Aria is evaluated across three core dimensions:


Authenticity — Does the response sound like a real, experienced therapist or like a chatbot performing therapy?
Clinical Accuracy — Does Dr. Aria apply the right approach for the given emotional context?
Non-AI Behaviour — Does it avoid LLM tells: over-validation, unsolicited advice, bullet lists, a question at the end of every message?


Each test case is scored out of 10. A score of 8–10 is a pass. A score of 6–7 is marginal and warrants review. Below 6 is a fail.

	How to Use This Scorecard
Follow these steps for each test case:


1.  Start the CLI chat client:
    npm run chat


2.  Type each "User" message exactly as written in the test case table.
3.  Copy Dr. Aria's response verbatim into the response area of the table.
4.  Score each of the 5 criteria (0, 1, or 2) and note which specific phrases triggered any deductions.
5.  Sum the criteria scores for the total (maximum 10).
