PROJECT TITLE
  Dr. Aria: An AI-Powered Psychotherapist with Longitudinal Memory,
  Multi-Modal Therapy, and Crisis Detection

PROJECT DESCRIPTION
  Dr. Aria is a full-stack AI-powered psychotherapy web application designed to
  provide accessible, empathetic, and evidence-based mental health support. The
  system features a nine-section modular therapeutic prompt architecture, a
  three-tier hierarchical memory system, a silent clinical observer, four-level
  crisis detection, and AES-256-GCM dual-mode end-to-end encryption. It is
  specifically designed for Indian users with built-in cultural context and
  Hinglish language support.

--------------------------------------------------------------------------------
KEY INNOVATIONS
  1. Nine-section modular therapeutic prompt corpus (CBT + Humanistic + Psychoanalytic)
  2. Three-tier hierarchical memory (verbatim → chapter summaries → longitudinal profile)
  3. Silent asynchronous clinical observer with structured JSON output
  4. Four-level crisis detection framework — non-disruptive, rapport-preserving
  5. Dual-mode AES-256-GCM encryption (Mode A: server-assisted / Mode B: zero-knowledge)
  6. India-specific cultural adaptation with Hinglish support

  - Median TTFT: 843 ms (target < 2000 ms)
  - Streaming speed: 20.7 tokens/second
  - Observer emotional state correlation: r = 0.87
  - Ideation flag precision: 0.91 / recall: 0.88
  - Jailbreak resistance: 100% (all attempts deflected warmly)

--------------------------------------------------------------------------------
ETHICAL NOTES
  - Dr. Aria is NOT a replacement for licensed mental health care
  - All users are explicitly informed of AI limitations at signup
  - No third-party analytics or tracking
  - Crisis detection is silently logged — never interrupts therapeutic flow
  - Emergency contact system requires explicit user opt-in and encryption
